## Why I started this

I&apos;ve been collecting half-finished notes in scratch files for years. This site
is my attempt to give those notes a home — somewhere I can publish them and let
them improve over time.

## What you&apos;ll find here

- **Project write-ups** with the trade-offs I actually wrestled with
- **Short notes** on things I learned this week
- **Long-form** essays when an idea earns the length

## How this is built

This blog runs on Next.js with the App Router, Tailwind CSS v4, shadcn/ui, and
Velite for content. Every post is a `.mdx` file in `content/blog/`. New post?
Drop a file in that folder and you&apos;re done.

> The best blog is the one you actually update.

Stick around.
