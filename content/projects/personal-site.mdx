## Overview

The site you&apos;re looking at. Built to be the smallest reasonable footprint
for a portfolio + blog that I&apos;ll actually maintain.

## Highlights

- Static generation everywhere — every page is a static HTML file at build time.
- Content lives as MDX in `content/`. Velite gives me typed access at build time.
- Dark mode via `next-themes`, with no flash on load.
- Dynamic Open Graph images per page via `next/og`.

## Lessons

Keeping the dependency list short paid off. Every dep I removed up front is one
less thing to upgrade later.
